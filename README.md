# WebOS Simulation

A browser-based desktop OS simulation built with Next.js, React, Tailwind CSS, and Framer Motion.

## Features
- **Window Management**: Draggable, resizable, minimizable windows.
- **File System**: Virtual file system with persistence (localStorage).
- **Apps**:
  - **Snake**: Playable game.
  - **Terminal**: Simulated shell (ls, cd, cat, echo, etc.).
  - **File Manager**: Browse, create, delete files.
  - **Text Editor**: Create and edit text files.
  - **Settings**: Change wallpaper (presets or upload).
- **Customization**: Persistent wallpaper.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
